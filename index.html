Keyboard Keys

What has 104 plastic buttons that goes rarely noticed during computer usage? The average desktop keyboard. Frequently, computer users ignore keyboards and relegate these essential input devices as accessories. Moreover, most keyboard concerns would be de-prioritized until needed buttons invariably don't work or a key is stuck.

Some brand name keyboards have been marketed to last up to 15 million keystrokes. But this may be highly suspect, Ergopedia.ca, suggests that the average membrane keyboard last more like 1 million keystrokes. Consider that an average computer user presses 1-3 million keystrokes per year based on job description alone, thereafter, this average user would continually purchase a new keyboard between 1-5 years. Why the large variance?
